# Dataset Risk Note

Gate 0 is intentionally conservative.

DenPAR is the open-release core because it is public, CC-BY-4.0, and contains the
landmarks needed for periodontal bone-level measurement. It is still single-timepoint
data, so it supports controlled synthetic-pair stress tests, not clinical
longitudinal outcome claims.

The Mendeley bitewing caries dataset is public but CC-BY-NC-3.0. It is excluded
from the open benchmark and release. If used later, it must be quarantined as an
optional non-commercial annotator-disagreement study.

The periapical lesion radiograph dataset is public and CC-BY-4.0, so it is usable
as an external periapical-domain shift/confounding source. It is still
single-timepoint and lesion-classification oriented, so it should not be used to
claim clinical longitudinal periodontal change validation.
